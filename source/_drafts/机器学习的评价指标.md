# 混淆矩阵

![混淆矩阵](/images/pasted-103.png)

- 准确率：$accuracy = \frac{TP+TN}{TP+TN+FP+FN}$
- 精度：$precision = \frac{TP}{TP+FP}$
- 查全率：$recall = \frac{TP}{TP+FN}$
- F1：$F1 sorce = \frac{2*Precision*Recall}{Precision+Recall}$
